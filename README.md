# Note Taking App made with Flutter and Firebase
A simple note taking app made with Flutter and Firebase for backend services.

## Functionalities
- Firebase Firestore database for storing notes and users.
- Firebase Authenticaiton service.
- Search notes.
- Sort notes based on creation/modification date.
